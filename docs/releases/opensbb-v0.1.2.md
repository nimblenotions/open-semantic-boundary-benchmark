## Open Semantic Boundary Benchmark v0.1.2

**Zenodo archive release** for the frozen medication-adherence pilot bundle. Benchmark artifacts are unchanged from v0.1.1.

### What's in this release

- **Frozen pilot run:** `outputs/pilot_v2/` (100 personas · 630 test events · seed 42)
- **Nine lattice conditions** with oracle transforms in `data/transformed/`
- **Policies, schemas, split manifest:** `data/policies/`, `data/schemas/`, `data/ground_truth/split_manifest_v0.json`
- **Config:** `configs/pilot_v0.1.1.yaml`
- **Reproduction:** `make repro-smoke` (no Ollama required for headline metrics)
